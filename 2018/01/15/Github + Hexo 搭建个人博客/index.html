Github + Hexo 搭建 Blog

前言

之前购买了阿里云服务器，用HTML5+CSS+JS搭建了个人博客，但瓶颈在于服务器只有1M带宽，加载一张较大的图片都要好半天，于是便放弃了自己搭Blog的想法。某一天偶然看到Github Pages + Hexo可以搭建Blog，而且不限带宽，遂重启这一计划。
